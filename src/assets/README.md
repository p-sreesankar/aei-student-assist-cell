# Assets Folder

Place static assets here that are imported directly in your code:

- Logos
- Icons
- Background images
- Any images used in components

## Gallery Images

If you prefer to store gallery images locally (instead of using Google Drive links), create a `gallery/` subfolder here and add your images.

Example:
```
src/assets/
├── logo.png
├── og-image.png        ← Social sharing preview image (1200×630 px)
└── gallery/
    ├── event-1.jpg
    ├── event-2.jpg
    └── team-photo.jpg
```
