========================================
Key Directories to Know:
========================================

📂 src/data/
    ↳ The ONLY folder you need to edit to update content
    ↳ Each file has detailed instructions at the top
    ↳ No HTML/CSS knowledge required

📂 src/pages/
    ↳ One React component per page/route
    ↳ Rendered by App.jsx via React Router

📂 src/components/
    ↳ Shared UI components used across multiple pages
    ↳ Layout, Navbar, Footer, etc.

📂 public/
    ↳ Static files served as-is (not processed by Vite)
    ↳ Use for large images, favicon, CNAME file (custom domain)

========================================
How Content Flows:
========================================

1. You edit a data file (e.g., src/data/notices.js) → add a new notice
2. The corresponding page (e.g., src/pages/Notices.jsx) imports that data
3. The page renders the data using React components
4. You run 'npm run deploy' → site updates on GitHub Pages

========================================
Common Tasks:
========================================

Add a notice:           Edit src/data/notices.js
Add an event:           Edit src/data/events.js
Add a gallery photo:    Edit src/data/gallery.js
Update contact info:    Edit src/data/faculty.js
Change site colors:     Edit src/data/site-config.js (themeColors section)
Swap Google Form:       Edit src/data/site-config.js (grievanceFormUrl)
